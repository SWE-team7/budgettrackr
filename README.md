# Getting Started
## Prerequisites
For python dependencies: 
```
# install pip (python pkg manager)
$ sudo apt install python3-pip

# verify install
$ pip --version

# install BS4, requests, jinja2, lxml
$ pip install beautifulsoup4 requests Jinja2 lxml
```
